Lernwirksamer digitaler Unterricht

Diese App bündelt zentrale Studien und Forschungsübersichten zu digitalem Unterricht, Tabletklassen, Lernplattformen, H5P, Feedback, KI und alternativen Leistungsnachweisen. Im Mittelpunkt steht: Was zeigen Forschungsergebnisse und was folgt daraus konkret für guten Unterricht?

Orientierung & Eingrenzung

Merksatz:
Digitale Unterrichtsentscheidungen sollten nicht vom Tool ausgehen, sondern von der Lernhandlung: Was sollen Schülerinnen und Schüler besser verstehen, abrufen, erklären, prüfen, überarbeiten, reflektieren oder anwenden können?
Planungsfunktion:
Die Ampel im Planungsbereich ist eine heuristische Reflexionshilfe. Sie ersetzt keine fachliche Prüfung, macht aber fehlende Planungsbausteine sichtbar. Der Hauptzweck bleibt die Informationstabelle.
Nutzung:
Wählen Sie zuerst einen Weg: Studien ansehen, eine Unterrichtsidee prüfen oder Quellen öffnen. Einzelne Studienhinweise können für den Planungscheck markiert werden.
Datenschutz / Speicherung:
Markierte Studienhinweise werden nur lokal in diesem Browser gespeichert. Es werden keine Eingaben übertragen. Beim Browserwechsel, Löschen der Browserdaten oder anderem Gerät kann die Auswahl verloren gehen.

Kurze Gebrauchsanweisung

Sie müssen die App nicht von oben nach unten durcharbeiten. Wählen Sie zuerst, was Sie gerade brauchen: eine schnelle Orientierung, die Prüfung einer konkreten Unterrichtsidee oder den Blick in die Quellen.

OrientierenBleiben Sie auf „Studien ansehen“, wählen Sie links ein Themenfeld oder einen Filter und lesen Sie die passenden Studienhinweise.
PlanenMarkieren Sie passende Studienhinweise und wechseln Sie zu „Unterrichtsidee prüfen“, um daraus einen Reflexionscheck zu erstellen.
BelegenÖffnen Sie „Quellenbasis“, wenn Sie Studien, Modelle oder Berichte nachschlagen möchten.

Im Mittelpunkt stehen nicht einzelne Tools, sondern die Lernhandlung: Was sollen Schülerinnen und Schüler besser verstehen, abrufen, erklären, prüfen, überarbeiten, reflektieren oder anwenden können?

1. Studien ansehen

Im Bereich „Studien ansehen“ finden Sie zentrale Forschungsergebnisse zu digitalem Unterricht, Lernplattformen, H5P, Feedback, KI, Gamification, Selbststeuerung und alternativen Leistungsnachweisen. Jeder Studienhinweis enthält eine kurze Einordnung, konkrete Konsequenzen für Unterricht und Quellenangaben.

Die Studienhinweise können über die Eingrenzungen „Modell / Bezugsrahmen“, „Unterrichtsfunktion“, „Tool- / Formatbezug“ und „Evidenz / Belastbarkeit“ gefiltert werden. Zusätzlich können Sie über die Themenfelder links schnell zu bestimmten Bereichen wechseln. Die frühere Freitextsuche ist bewusst entfernt.

Einzelne Studienhinweise können über „für Planungscheck markieren“ ausgewählt werden. Wurden Studienhinweise markiert, können sie über „Auswahl im Planungscheck nutzen“ in den Planungscheck übernommen werden. Alternativ stehen „Alle markieren“ und „Auswahl leeren“ zur Verfügung, um die Auswahl schnell anzupassen.

2. Planungs-Prompt zu angezeigten Studienhinweisen nutzen

Über „Prompt zu angezeigten Studienhinweisen kopieren“ wird ein gemeinsamer KI-Arbeitsauftrag zu den aktuell sichtbaren Einträgen erstellt. Das ist besonders sinnvoll, wenn die Studienhinweise vorher über Modell, Unterrichtsfunktion, Tool-/Formatbezug, Evidenz/Belastbarkeit oder Themenfeld eingegrenzt wurden.

Der Prompt enthält die ausgewählten Studienhinweise mit Einordnung, didaktischer Konsequenz, Umsetzungshinweisen und Quellenankern. Ergänzen Sie anschließend Fach, Klasse, Thema, Lernziel und Rahmenbedingungen. So kann aus mehreren passenden Forschungsergebnissen eine konkrete Unterrichtsidee für die eigene Lerngruppe entwickelt werden.

3. Unterrichtsidee prüfen

Im Bereich „Unterrichtsidee prüfen“ können Sie eine eigene Unterrichtsidee, digitale Aufgabe, Tabletphase, KI-Aufgabe, gamifizierter Lernpfad oder ein Projekt kurz eingeben, damit diese anhand der Kriterien des Planungschecks reflektiert und eingeschätzt werden kann.

Wählen Sie dazu das passende Format aus und bearbeiten Sie die Checkliste. Die App zeigt über eine Reflexionsampel, ob zentrale Planungsbausteine bereits sichtbar sind oder ob wichtige Punkte noch fehlen.

Über „Check erstellen“ erhalten Sie einen kompakten Reflexionscheck. Über „Check mit passenden Studienhinweisen erstellen“ werden fehlende Punkte zusätzlich mit passenden Forschungsergebnissen aus der Tabelle verknüpft. So erhalten Sie konkrete Hinweise, wie die Unterrichtsidee didaktisch geschärft werden kann.

4. Check weiterverwenden

Der erzeugte Check eignet sich z. B. für die eigene Unterrichtsplanung, die Besprechung im Team oder als Reflexionsgrundlage in Fortbildungen. Er kann über „Check kopieren“ in andere Dokumente übernommen oder über „Check als Markdown laden“ heruntergeladen werden.

Über „Prompt aus Check kopieren“ kann zusätzlich ein KI-Arbeitsauftrag erzeugt werden, der die eingegebene Unterrichtsidee, offene Prüfpunkte, markierte Studienhinweise und passende Forschungsergebnisse zur Nachschärfung zusammenführt. Dieser Prompt ist besonders sinnvoll, wenn aus dem Reflexionscheck eine konkretere Unterrichtsplanung entstehen soll.

5. Quellenbasis

Im Bereich „Quellenbasis“ finden Sie die verwendeten Studien, Reviews, Modelle und Berichte mit kurzer Einordnung und Link.

Die Quellenangaben dienen der Orientierung. Die Angabe „stark/moderat/vorsichtig“ meint hier die Belastbarkeit der didaktischen Ableitung, nicht eine numerische Effektstärke.

Hinweis zur Speicherung

Markierte Studienhinweise werden nur lokal im jeweiligen Browser gespeichert. Es werden keine Eingaben übertragen. Beim Löschen der Browserdaten, Browserwechsel oder Gerätewechsel kann die Auswahl verloren gehen.

Die kopierten Prompts sind bewusst als Arbeitsaufträge formuliert. Prüfen Sie KI-Ausgaben fachlich und didaktisch, bevor Sie sie im Unterricht einsetzen.

Kurze Erklärung nötig?

Diese App ist kein Tool-Katalog und kein automatischer Unterrichtsbewerter. Sie hilft dabei, digitale Unterrichtsideen anhand von Studien und Forschungsergebnissen zu planen und zu prüfen.

1. Studien ansehenNutzen Sie die Themenfelder und Filter, um passende Forschungshinweise zu finden. Es geht um Lernhandlungen, nicht um möglichst viele digitale Werkzeuge.
2. Unterrichtsidee prüfenWenn Sie eine konkrete Idee haben, wechseln Sie zum Planungscheck, wählen das Format aus und bearbeiten die Prüfpunkte.
3. Ergebnis nutzenDer Check zeigt offene Punkte, passende Studienhinweise und nächste Schritte. Die Ampel ist eine Reflexionshilfe, keine automatische Bewertung.
